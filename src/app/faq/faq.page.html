FAQ

What information that provided in this application?

This mobile application provide many authentic information about common disease. You can know how to avoid the problem, how it can be happening and many more.

How many common diseases in this application?

So far, this application have around 21 common of disease that will help people to solve their problems.

How to know the solution of each diseases?

You can just search or scrolling the information by clicking the next button. You can see each diseases have their own grid button which one of them is the solution for diseases that you want to know.

Did this application provide by having a consultation with the doctor?

Unfortunately, this application did not provide this kind of question. But, this will be a good suggestions for developer to make this information in the future.
